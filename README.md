# USAF-proyecto-IA-DATA-BASE
Este codigo / programa tiene almacenado datos interesantes de modelos de avioens de la USAF // This code/program contains stored interesting data about aircraft models of the USAF.


Este programa es una base de conocimientos interactiva sobre aviones militares, principalmente de la USAF. Permite a los usuarios buscar información detallada sobre aviones específicos (como su rol, velocidad, motores y datos históricos) utilizando búsquedas parciales e insensibles a mayúsculas/minúsculas. Si encuentra múltiples coincidencias, te pedirá que seas más específico.

This program is an interactive knowledge base about military aircraft, primarily from the USAF. It allows users to search for detailed information on specific aircraft (such as their role, speed, engines, and historical facts) using partial and case-insensitive searches. If multiple matches are found, it will prompt you to be more specific.
